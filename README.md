# Astrofysische_Simulaties
main repository of project

Random uniform creert een random double tussen 0 en 1. Deze functie wordt gebruikt door RandomVec en Randommass om willekeurige vectoren en massa's te genereren. Daarna worden deze herschaalt zodat we altijd een gebonden systeem hebben en zodat de totale massa altijd 1 is.
